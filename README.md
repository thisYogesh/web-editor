## To run on dev environment
```
yarn backend-dev
yarn frontend-dev
yarn dev
```

## To run on production environment
```
yarn backend-build
yarn frontend-build
yarn serve
```


## (webitor) NPM package directory structure

```
├── dist
│   ├── backend
│   ├── frontend
├── css
├── fonts
├── index.js
├── package.json
```
